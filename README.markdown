jviotti.com
===========

My personal website.

Running locally
---------------

```sh
make serve
```

Write a new article
-------------------

1. Create Markdown file at `articles/YYYY-MM-DD-title.markdown`
2. Put any images, if any, in `images/<name>.{png,jpg}`
3. Update `content.mk` to list the article
4. Update `content.mk` to list the images, if any
